rpm-builder:deploy

Full name:

org.jboss.pnc:rpm-builder-maven-plugin:1.4:deploy

Description:

Deploy the built rpms to a specified repository.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Binds by default to the lifecycle phase: deploy.

Optional Parameters

Name Type Since Description
<rpmDeploymentRepository> String - Target URI deployment repository. This should support the same format as altDeploymentRepository. Falls back to the value of the user property altDeploymentRepository.
User Property: rpmDeploymentRepository
<skip> boolean - Whether to skip the plugin
Default: false
User Property: rpm.deploy.skip
